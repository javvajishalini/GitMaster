## 📖 About The Project

**GitMaster** is a modern, responsive, and highly interactive educational platform designed to help beginners master Git. It goes beyond static tutorials by offering an embedded **Simulated Git Terminal** right in the browser. Users can practice real Git commands, learn fundamental concepts, test their knowledge with quizzes, and track their progress—all within a stunning "liquid glass" UI.

This project is actively participating in **GirlScript Summer of Code (GSSoC)**!

## ✨ Features

- **Interactive Git Terminal**: A browser-based terminal emulator that parses and executes basic Git commands (`init`, `add`, `commit`, `status`, `log`), providing realistic feedback.
- **Structured Tutorials**: Step-by-step lessons covering the essential Git lifecycle from initialization to branching.
- **Quiz Arena**: Built-in multiple-choice quizzes with instant feedback and explanatory notes.
- **Command Reference**: A searchable cheat sheet of common Git commands for quick lookup.
- **Progress Tracking**: Automatic progress saving via `localStorage`, complete with visual progress bars.
- **Stunning UI/UX**: Built with a custom Tailwind CSS v4 design system featuring dark mode, glassmorphism, and micro-animations.

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/) (bootstrapped with [Vite](https://vitejs.dev/))
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Context API & LocalStorage

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm (Node Package Manager)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/javvajishalini/GitMaster.git
   ```
2. **Navigate into the project directory:**
   ```bash
   cd GitMaster
   ```
3. **Install the dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
5. **Open your browser** and visit `http://localhost:5173`.
